# Wildfire
Wildfire Detection and Prediction AI

### dataloader.py
Used to convert the custom datasets into dataloaders ready for model training and evaluations.

### models.py
Contains all the models that require training

### trainer.py
Used as a helper to train the models in `models.py` using the dataloaders from `dataloaders.py`.

### train.py
Used to train the models on different dataloaders.
